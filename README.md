## Abhayagiri's 20th Anniversary

Source for https://www.abhayagiri.org/20.

## Setup

```sh
git clone https://github.com/abhayagiri/abm-20th-anniversary
cd abm-20th-anniversary
npm install
bower install
```

Add the media to `dist/DVD`.

# Build

```sh
gulp media # Build dist/DVD/Web/img
gulp manifest # Build audio and photos manifest
gulp compile # Build everything else
```
